# invs

## Installation
```
go get github.com/kozakana/invs
```

## Usage

```
$ invs <options>
```

## Options
* `-p`: Port number to listen on. Defaults to 8080.
* `--filter-url`: Output only URLs that contain the string.
* `--proxy-host`: Proxy to the option value.(ex. localhost:9999)

## Build
```
go build invs.go
```
